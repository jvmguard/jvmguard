The jvmguard API is used for Declared transactions an telemetries.

See the package descriptions for more information.

Manual download

The jvmguard API is located in the JAR file api/jvmguard-api.jar in the jvmguard installation directory (download here).

Maven 


  <dependency>
     <groupId>com.jvmguard</groupId>
     <artifactId>api</artifactId>
     <version>@VERSION@</version>
  </dependency>

Gradle 


  dependencies {
      implementation 'com.jvmguard:api:@VERSION@'
  }
