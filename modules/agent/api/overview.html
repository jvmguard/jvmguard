The jvmguard API is used for Declared transactions and telemetries.

See the package descriptions for more information.

Manual download

The jvmguard API is located in the JAR file api/jvmguard-annotations.jar in the jvmguard installation directory (download here).

Maven 


  <dependency>
     <groupId>dev.jvmguard</groupId>
     <artifactId>jvmguard-annotations</artifactId>
     <version>@VERSION@</version>
     <scope>provided</scope>
  </dependency>

Gradle 


  dependencies {
      compileOnly 'dev.jvmguard:jvmguard-annotations:@VERSION@'
  }
